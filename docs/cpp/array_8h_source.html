456 // A dictionary array contains an array of non-negative integers (the
457 // "dictionary indices") along with a data type containing a "dictionary"
458 // corresponding to the distinct values represented in the data.
459 //
460 // For example, the array
461 //
462 // ["foo", "bar", "foo", "bar", "foo", "bar"]
463 //
464 // with dictionary ["bar", "foo"], would have dictionary array representation
465 //
466 // indices: [1, 0, 1, 0, 1, 0]
467 // dictionary: ["bar", "foo"]
468 //
469 // The indices in principle may have any integer type (signed or unsigned),
470 // though presently data in IPC exchanges must be signed int32.
