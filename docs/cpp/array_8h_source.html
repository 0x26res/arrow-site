696 // "dictionary indices") along with a data type containing a "dictionary"
697 // corresponding to the distinct values represented in the data.
698 //
699 // For example, the array
700 //
701 // ["foo", "bar", "foo", "bar", "foo", "bar"]
702 //
703 // with dictionary ["bar", "foo"], would have dictionary array representation
704 //
705 // indices: [1, 0, 1, 0, 1, 0]
706 // dictionary: ["bar", "foo"]
707 //

709 // though presently data in IPC exchanges must be signed int32.
