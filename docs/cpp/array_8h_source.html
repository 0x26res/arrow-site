643 // A dictionary array contains an array of non-negative integers (the
644 // "dictionary indices") along with a data type containing a "dictionary"
645 // corresponding to the distinct values represented in the data.
646 //
647 // For example, the array
648 //
649 // ["foo", "bar", "foo", "bar", "foo", "bar"]
650 //
651 // with dictionary ["bar", "foo"], would have dictionary array representation
652 //
653 // indices: [1, 0, 1, 0, 1, 0]
654 // dictionary: ["bar", "foo"]
655 //
656 // The indices in principle may have any integer type (signed or unsigned),
657 // though presently data in IPC exchanges must be signed int32.
