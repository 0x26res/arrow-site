609 // A dictionary array contains an array of non-negative integers (the
610 // "dictionary indices") along with a data type containing a "dictionary"
611 // corresponding to the distinct values represented in the data.
612 //
613 // For example, the array
614 //
615 // ["foo", "bar", "foo", "bar", "foo", "bar"]
616 //
617 // with dictionary ["bar", "foo"], would have dictionary array representation
618 //
619 // indices: [1, 0, 1, 0, 1, 0]
620 // dictionary: ["bar", "foo"]
621 //
622 // The indices in principle may have any integer type (signed or unsigned),
623 // though presently data in IPC exchanges must be signed int32.
