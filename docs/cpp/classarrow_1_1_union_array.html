static

Construct Dense UnionArray from types_ids, value_offsets and children.

This function does the bare minimum of validation of the offsets and input types. The value_offsets are assumed to be well-formed.

Parameters
[in]type_idsAn array of 8-bit signed integers, enumerated from 0 corresponding to each type.
[in]value_offsetsAn array of signed int32 values indicating the relative offset into the respective child array for the type in a given slot. The respective offsets for each child value array must be in order / increasing.
[in]childrenVector of children Arrays containing the data for each type.
[out]outWill have length equal to value_offsets.length()
