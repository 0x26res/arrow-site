Detailed Description

A reusable block-based chunker for CSV data.

The chunker takes a block of CSV data and finds a suitable place to cut it up without splitting a row. If the block is truncated (i.e. not all data can be chunked), it is up to the caller to arrange the next block to start with the trailing data.

Note: if the previous block ends with CR (0x0d) and a new block starts with LF (0x0a), the chunker will consider the leading newline as an empty line.
