Detailed Description

template<typename DType>
class parquet::DictEncoder< DType >

See the dictionary encoding section of https://github.com/Parquet/parquet-format.

The encoding supports streaming encoding. Values are encoded as they are added while the dictionary is being constructed. At any time, the buffered values can be written out with the current dictionary size. More values can then be added to the encoder, including new dictionary entries.

Writes out any buffered indices to buffer preceded by the bit width of this data.

Returns the number of bytes written. If the supplied buffer is not big enough, returns -1. buffer must be preallocated with buffer_len bytes. Use EstimatedDataEncodedSize() to size buffer.
