class pyarrow.Table¶

Bases: object

A collection of top-level named, equal length Arrow arrays.

Warning

Do not call this class’s constructor directly, use one of the from_* methods instead.

from_pandas(type cls, df, bool timestamps_to_ms=False, Schema schema=None, bool preserve_index=True)¶

Convert pandas.DataFrame to an Arrow Table

Parameters:
  • df (pandas.DataFrame) –
  • timestamps_to_ms (bool) –

    Convert datetime columns to ms resolution. This is needed for compability with other functionality like Parquet I/O which only supports milliseconds.

    Deprecated since version 0.7.0.

  • schema (pyarrow.Schema, optional) – The expected schema of the Arrow Table. This can be used to indicate the type of columns if we cannot infer it automatically.
  • preserve_index (bool, optional) – Whether to store the index as an additional column in the resulting Table.
Returns:

pyarrow.Table

Examples

>>> import pandas as pd
>>> import pyarrow as pa
>>> df = pd.DataFrame({
    ...     'int': [1, 2],
    ...     'str': ['a', 'b']
    ... })
>>> pa.Table.from_pandas(df)
