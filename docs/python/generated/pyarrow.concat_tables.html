pyarrow.concat_tables¶

pyarrow.concat_tables(tables)¶

Perform zero-copy concatenation of pyarrow.Table objects. Raises exception if all of the Table schemas are not the same

Parameters:
  • tables (iterable of pyarrow.Table objects) –
  • output_name (string, default None) – A name for the output table, if any
