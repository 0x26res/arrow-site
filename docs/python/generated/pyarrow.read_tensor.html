pyarrow.read_tensor¶

pyarrow.read_tensor(NativeFile source)¶

Read pyarrow.Tensor from pyarrow.NativeFile object from current position. If the file source supports zero copy (e.g. a memory map), then this operation does not allocate any memory

Parameters:source (pyarrow.NativeFile) –
Returns:tensor (Tensor)
